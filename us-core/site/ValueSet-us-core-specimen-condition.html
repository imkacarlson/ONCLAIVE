ValueSet: US Core Specimen Condition

Official URL: http://hl7.org/fhir/us/core/ValueSet/us-core-specimen-condition Version: 8.0.0
Standards status: Trial-use Maturity Level: 2 Computable Name: USCoreSpecimenCondition
Other Identifiers: OID:2.16.840.1.113883.4.642.40.2.48.15

Copyright/Legal: This value set includes content from SNOMED CT, which is copyright © 2002+ International Health Terminology Standards Development Organisation (IHTSDO), and distributed by agreement between IHTSDO and HL7. Implementer use of SNOMED CT is not covered by this agreement

This value set contains concepts the provide information regarding a specimen, including the container, that does not meet a laboratory’s criteria for acceptability.
