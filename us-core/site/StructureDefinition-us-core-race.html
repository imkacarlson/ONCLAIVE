Extension: US Core Race Extension

Official URL: http://hl7.org/fhir/us/core/StructureDefinition/us-core-race Version: 8.0.0
Standards status: Trial-use Maturity Level: 3 Computable Name: USCoreRaceExtension
Other Identifiers: OID:2.16.840.1.113883.4.642.40.2.42.54

Copyright/Legal: Used by permission of HL7 International, all rights reserved Creative Commons License

Concepts classifying the person into a named category of humans sharing common history, traits, geographical origin or nationality. The race codes used to represent these concepts are based upon the CDC Race and Ethnicity Code Set Version 1.0 which includes over 900 concepts for representing race and ethnicity of which 921 reference race. The race concepts are grouped by and pre-mapped to the 5 OMB race categories:

  • American Indian or Alaska Native
  • Asian
  • Black or African American
  • Native Hawaiian or Other Pacific Islander
  • White.

Complies with 2015 Edition Common Clinical Data Set for patient race.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: Patient
    • Element ID: RelatedPerson
    • Element ID: Person
    • Element ID: Practitioner
    • Element ID: FamilyMemberHistory

Profile specific implementation guidance

  • This Complex Extension for race allows one or more race codes of which:
    • Must Support at least one code from OMB Race Categories
    • May include additional race codes from CDC Race Codes
    • Must include a text description of race
  • A race search parameter has been defined for this extension. However, there are no race search requirements for the Client or Server.

Usage info

Usages:

  • Use this Extension: US Core Patient Profile
  • Examples for this Extension: Patient/child-example, Patient/deceased-example, Patient/example and Patient/infant-example
